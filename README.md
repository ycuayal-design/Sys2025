## Organización del repositorio
- `paciales/` (parciales): subcarpetas `1/` y `2/` para el material y las soluciones de cada parcial.
- `talleres/`: cuadernos de trabajo en Jupyter (`*.ipynb`) con ejercicios, talleres guiados y experimentos (p. ej. `taller sympy.ipynb`, `taller_2partes2_1y2_5.ipynb`).
- `otros/`: material de referencia y ejemplos adicionales (`Copia_de_ejemplo1SyS2021.ipynb`, `Copia_de_transformada_de_fourier.ipynb`, etc.).

## Cómo trabajar con los cuadernos
1. Crea o activa un entorno de Python con Jupyter instalado.
2. Desde la raíz del repositorio ejecuta `jupyter notebook` o `jupyter lab`.
3. Abre los cuadernos dentro de `talleres/`, `paciales/` o `otros/` según corresponda y ejecuta las celdas.
